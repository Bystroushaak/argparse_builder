argparse builder

Python 2.7 argparse builder. This website helps you to quickly create complex argparse interfaces.

ArgumentParser objects

prog

The name of the program. Default sys.argv[0].

usage

The string describing the program usage (default: generated from arguments added to parser).

description

Text to display before the argument help (default: None). It shouldn't be too long - don't mistake this with man page.

epilog

Text to display after the argument help (default: None).
